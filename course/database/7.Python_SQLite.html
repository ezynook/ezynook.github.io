SQLAlchemy คืออะไร?

SQLAlchemy เป็นไลบรารีภาษา Python ที่ใช้สำหรับโต้ตอบกับฐานข้อมูลเชิงสัมพันธ์ (Relational Database) โดยมีจุดเด่นคือช่วยให้นักพัฒนาสามารถทำงานกับฐานข้อมูลได้ง่ายขึ้น โดยไม่ต้องเขียน SQL โดยตรงมากนัก

ทำไมต้องใช้ SQLAlchemy?

  • ORM (Object-Relational Mapper): SQLAlchemy ทำให้เราสามารถมองตารางในฐานข้อมูลเป็น object ใน Python ได้ ทำให้การเขียนโค้ดเพื่อจัดการข้อมูลเป็นไปได้อย่างเป็นธรรมชาติ
  • ยืดหยุ่น: รองรับฐานข้อมูลหลากหลายชนิด เช่น PostgreSQL, MySQL, SQLite, Oracle และอื่นๆ
  • มีประสิทธิภาพ: สามารถเขียน query ที่ซับซ้อนได้อย่างง่ายดาย และมีเครื่องมือสำหรับ optimize query เพื่อเพิ่มประสิทธิภาพ
  • ใช้งานง่าย: มีเอกสารประกอบที่ครบถ้วน และมี community ที่คอยช่วยเหลือ

Pandas คืออะไร?

Pandas เป็นไลบรารีภาษา Python ที่ได้รับความนิยมอย่างมากสำหรับการจัดการและวิเคราะห์ข้อมูล โดยเฉพาะข้อมูลที่อยู่ในรูปแบบตารางคล้ายกับ Excel หรือฐานข้อมูลเชิงสัมพันธ์ (Relational Database) คิดง่ายๆ ว่า Pandas เป็นเหมือนเครื่องมือขั้นสูงที่ช่วยให้เราทำงานกับข้อมูลได้สะดวกและรวดเร็วขึ้น

ทำไมต้องใช้ Pandas?

  • จัดการข้อมูลได้ง่าย: Pandas ช่วยให้เราสามารถอ่านข้อมูลจากไฟล์ต่างๆ (เช่น CSV, Excel) มาจัดเก็บในโครงสร้างข้อมูลที่เรียกว่า DataFrame ซึ่งคล้ายกับตารางใน Excel ทำให้เราสามารถเข้าถึงข้อมูลแต่ละส่วนได้อย่างง่ายดาย

  • วิเคราะห์ข้อมูลเบื้องต้น: Pandas มีฟังก์ชันมากมายที่ช่วยในการวิเคราะห์ข้อมูลเบื้องต้น เช่น การหาค่าสถิติ (mean, median, standard deviation), การกรองข้อมูล, การจัดกลุ่มข้อมูล, และการรวมข้อมูล

  • เตรียมข้อมูลสำหรับ Machine Learning: ก่อนที่จะนำข้อมูลไปใช้ในการสร้างโมเดล Machine Learning เราจำเป็นต้องทำการเตรียมข้อมูลให้พร้อม Pandas ช่วยให้เราทำขั้นตอนนี้ได้อย่างมีประสิทธิภาพ

  • ทำงานร่วมกับไลบรารีอื่นๆ ได้ดี: Pandas สามารถทำงานร่วมกับไลบรารีอื่นๆ ในระบบนิเวศของ Python ได้อย่างราบรื่น เช่น NumPy, Matplotlib, Seaborn ซึ่งช่วยให้เราสามารถสร้างกราฟและวิเคราะห์ข้อมูลเชิงลึกได้

from sqlalchemy import create_engine
import pandas as pd

engine = create_engine(
    "sqlite:///demo.db"
)

with engine.connect() as conn:
    query = "" #SQL
    result = conn.execute(query)
    df = pd.DataFrame(
        result.fetchall(), 
        columns=result.keys()
    )
    print(df)
