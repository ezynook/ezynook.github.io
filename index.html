เรียน Docker Step by Step ครบจบในคอร์สเดียว

Docker คือ Software Container แบบใหม่ที่ใช้งานได้ง่ายกว่า เมื่อเทียบกับ Container ตัวอื่นๆทำให้ได้รับความนิยมในวงกว้าง และกำลังเข้ามามีบทบาทในกลุ่ม Developer และ DevOps หรือ System Admin มากขึ้นเรื่อยๆ

สนใจเรียน


สร้าง API ด้วยภาษา FastAPI

Web Framework ที่พัฒนาด้วย Python เหมือนกันที่ชื่อว่า FastAPI ซึ่งเป็นอีกหนึ่งตัวที่น่าสนใจในการนำมาสร้าง API โดยตัวนี้ออกแบบมาให้ง่ายต่อการพัฒนาเป็นอย่างมากเราสามารถที่จะสร้าง API ขึ้นมาได้อย่างรวดเร็ว เข้าใจได้ง่ายและถ้าพูดถึงความเร็วในการใช้งานเชิงประสิทธิภาพนั้นเรียกได้ว่าเร็วเทียบเท่าตัว NodeJS และ Go ได้เลย

สนใจเรียน


การเขียนโปรแกรมเพื่อพัฒนา web application ด้วยภาษา PHP

เรียนรู้การพัฒนา Web Application ด้วยภาษา PHP ที่นิยมที่สุดในปัจจุบัน มีโครงสร้างภาษาที่ง่ายสำหรับผู้เริ่มต้น แต่มีประสิทธิภาพสูงขั้นมืออาชีพ อีกทั้งยังทำงานได้กับทุก Platform ไม่ว่าจะเป็น Windows, Linux หรือ Unix

สนใจเรียน


จัดการ Server ด้วย Linux (Linux Administrator)

Linux คือ ระบบปฏิบัติการ (Operating System) ประเภทหนึ่งเช่นเดียวกับ Windows หรือ Unix เหมาะสำหรับการทำงานแบบ Server และ Network นิยมใช้เป็น Application Server ต่างๆ

สนใจเรียน

จัดการข้อมูลขนาดใหญ่ด้วย Apache Hadoop (Big Data)

Hadoop เป็นซอฟต์แวร์ ตัวหนึ่งที่นิยมนำมาใช้ในการทำ Big Data เนื่องจาก Hadoop นั้นเป็น Open Source สามารถที่จะพัฒนาต่อยอดได้ และมีเครื่องมือต่าง ๆ ที่สามารถมาเชื่อมต่อกับ Hadoop เพื่อใช้งานได้ เช่น Pig, Hive, Hbase เป็นต้น โดยใช้ Hardware PC หรือ Server ทั่วไปมาสร้างเป็น Hadoop Cluster ได้
